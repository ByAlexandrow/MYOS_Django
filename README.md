### Mystery Of Story

Mystery of Story - уникальные истории всех жанров для всех возрастов))

1. Python

3. JavaScript, HTML5, CSS3 + адаптация для мобильный устройств
4. TinyMCE
5. Pytest + обработчик 500, 404, 403
